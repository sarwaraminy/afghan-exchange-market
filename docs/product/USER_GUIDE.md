# Afghan Exchange Market - User Guide

## Getting Started

### Creating an Account

1. Navigate to the application
2. Click **Register** or **Sign Up**
3. Fill in the registration form:
   - **Username**: 3-30 characters
   - **Email**: Valid email address
   - **Password**: Must contain:
     - At least 8 characters
     - One uppercase letter (A-Z)
     - One lowercase letter (a-z)
     - One number (0-9)
     - One special character (!@#$%^&*)
4. Click **Register**
5. You will be automatically logged in

### Logging In

1. Click **Login**
2. Enter your email and password
3. Click **Sign In**
4. You will be redirected to the Dashboard

### Changing Language

1. Look for the language selector in the header
2. Choose from:
   - **English** (EN)
   - **دری** (Dari)
   - **پښتو** (Pashto)
3. The interface will update immediately

## Viewing Exchange Rates

### Exchange Rates Page

1. Click **Rates** in the navigation
2. View rates for all currencies
3. Use the market filter to see rates from specific markets:
   - Sarai Shahzada (Kabul)
   - Khorasan Market (Herat)
   - Da Afghanistan Bank

### Understanding Rate Display

| Column | Description |
|--------|-------------|
| Currency | Currency code and name |
| Buy Rate | Rate when buying the currency |
| Sell Rate | Rate when selling the currency |
| Change | Percentage change from previous rate |

### Rate Indicators

- **Green arrow (↑)**: Rate increased
- **Red arrow (↓)**: Rate decreased
- **Gray (-)**: No change

## Currency Converter

### Converting Currencies

1. Click **Converter** in the navigation
2. Select **From** currency
3. Select **To** currency
4. Enter the **Amount**
5. (Optional) Select a specific **Market**
6. View the converted result

### Example

Converting 100 USD to AFN:
- From: USD (US Dollar)
- To: AFN (Afghan Afghani)
- Amount: 100
- Result: 7,050 AFN (at rate 70.50)

## Managing Favorites

### Adding a Currency to Favorites

1. Go to **Rates** page
2. Find the currency you want to favorite
3. Click the **star** icon next to it
4. The currency is now in your favorites

### Viewing Favorites

1. Go to **Dashboard**
2. Your favorite currencies appear in the Favorites section
3. See quick rates for all favorites

### Removing from Favorites

1. Go to **Dashboard** or **Rates**
2. Click the **filled star** icon on a favorited currency
3. The currency is removed from favorites

## Setting Price Alerts

### Creating an Alert

1. Go to **Dashboard**
2. Click **Add Alert** or navigate to Alerts section
3. Fill in the alert form:
   - **Currency**: Select the currency to monitor
   - **Target Rate**: The rate that triggers the alert
   - **Alert Type**:
     - **Above**: Alert when rate goes above target
     - **Below**: Alert when rate goes below target
4. Click **Create Alert**

### Managing Alerts

#### View All Alerts
- Go to Dashboard > Alerts section
- See all your active and triggered alerts

#### Edit an Alert
1. Find the alert in your list
2. Click the **Edit** icon
3. Modify the target rate or type
4. Save changes

#### Delete an Alert
1. Find the alert in your list
2. Click the **Delete** icon
3. Confirm deletion

#### Enable/Disable Alert
- Toggle the switch next to any alert to enable or disable it

## Troubleshooting

### Common Issues

#### Can't Log In
- Verify your email is correct
- Check your password (case-sensitive)
- Try resetting your password

#### Rates Not Updating
- Refresh the page
- Check your internet connection
- Try clearing browser cache

#### Language Not Changing
- Refresh the page after selecting language
- Clear browser cookies if issue persists

#### Alerts Not Working
- Verify the alert is enabled
- Check the target rate is realistic
- Ensure you're logged in

### Getting Help

If you encounter issues:
1. Try refreshing the page
2. Clear your browser cache
3. Try a different browser
4. Contact support with details of the issue

## Tips & Best Practices

### For Traders

1. **Set Multiple Alerts**: Create alerts for both above and below thresholds
2. **Use Favorites**: Add currencies you trade frequently
3. **Compare Markets**: Check rates across different markets for best deals
4. **Check Regularly**: Exchange rates can change rapidly

### For Businesses

1. **Monitor Key Currencies**: Set up alerts for currencies relevant to your business
2. **Track Gold Prices**: If dealing in jewelry or gold trade
3. **Read Announcements**: Stay updated on regulatory changes

### Security Tips

1. **Use Strong Passwords**: Follow the password requirements
2. **Log Out**: Always log out on shared computers
3. **Don't Share Credentials**: Keep your login private
4. **Check URL**: Ensure you're on the official site
